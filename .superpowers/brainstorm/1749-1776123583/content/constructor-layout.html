A. Трёхпанельный (sidebar + main + bottom)

Классический desktop layout. Список тем слева, вопросы выбранной темы в центре, блиц внизу. Toolbar с кнопками экспорта/импорта наверху или внизу.

Pros

  • Всё видно одновременно
  • Быстрая навигация между темами

Cons

  • На мобильных нужен другой layout
  • Сложнее адаптировать

B. Табы (темы / блиц / настройки)

Одноколоночный layout с табами наверху. Таб «Темы» показывает chips тем + вопросы выбранной темы. Таб «Блиц» — блиц-задания. Таб «Настройки» — API key и параметры AI.

Pros

  • Одинаково работает на mobile и desktop
  • Простая навигация

Cons

  • Нельзя видеть темы и блиц одновременно
  • Больше переключений

C. Аккордеон (всё на одной странице)

Одна длинная страница. Блок тем наверху (chips), затем collapsible-секции по каждой теме с вопросами, внизу — блиц-задания тоже collapsible. Всё видно в структуре, раскрываешь нужное.

Pros

  • Видна вся структура целиком
  • Хорошо на mobile
  • Нет скрытых панелей

Cons

  • При многих темах — длинный скролл
  • Менее «приложение», более «документ»
