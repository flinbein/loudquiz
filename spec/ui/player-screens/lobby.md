# Player: Lobby

Экран игрока на фазе [lobby](../../game/phases.md#lobby).
Отображается на мобильном устройстве игрока.

## Подключение

Перед входом в комнату игрок:

1. Переходит по URL (из QR-кода или по ссылке).
2. Вводит **имя** — предзаполнено из localStorage.

Имя игрока является его **уникальным ID**. Если игрок с таким именем
уже подключён — предыдущее подключение отключается.

## Элементы экрана

### Выбор команды

В режиме двух команд — кнопки выбора команды (красная / синяя).
В режиме одной команды — автоматическое распределение.

### Аватар

[PlayerAvatar](../components/PlayerAvatar.md) текущего игрока.
Клик по аватару — **смена иконки** на случайную из свободных.

Подробнее: [аватары](../avatars.md).

### Кнопки

- **Калибровка** — открывает [попап калибровки](../../systems/audio.md).
- **Готов** — фиксирует выбор игрока (команда, аватар). После нажатия
  игрок не может менять настройки.
- **Начать** — отображается вместо «Готов», когда все игроки готовы.
  Любой игрок может начать игру.

## Переподключение

Если игрок отключился:
- Его статус в игре меняется на «offline».
- Игрок может переподключиться и продолжить игру.

После начала игры новые игроки **не допускаются**.
