# Генерация тем

AI выбирает темы для викторины на основе предложений игроков.
Используется на фазе [topics-suggest](../../game/phases.md#topics-suggest).

## Входные данные

```typescript
interface TopicGenerationInput {
  suggestions: Array<{
    playerName: string;
    text: string;
  }>;
  topicCount: number;
  pastTopics: string[];
}
```

| Поле | Описание |
|---|---|
| `suggestions` | Предложения игроков (имя + текст) |
| `topicCount` | Сколько тем нужно выбрать |
| `pastTopics` | Темы, которые уже участвовали в прошлых играх |

## Правила для AI

### System prompt

Роль: ведущий викторины, выбирающий темы.

Инструкции:
- Выбрать `topicCount` тем, учитывая **пожелания большинства** игроков.
- **Игнорировать** непонятные или нецензурные предложения.
- **Не выбирать** узкоспециализированные темы (квантовая физика,
  молекулярная биология и т.д.).
- **Не повторять** темы из `pastTopics`.
- Если предложений мало — **предложить свои** варианты.

### User prompt

Содержит:
- Список предложений игроков.
- Количество тем для выбора.
- Список прошлых тем (если есть).

## Результат

```typescript
interface TopicGenerationResult {
  topics: Array<{
    name: string;    // название темы
    reason: string;  // почему выбрана эта тема
  }>;
}
```

Результат отображается ведущему, который может:
- Удалить тему из списка.
- Добавить свою тему.
- Принять список и перейти к [генерации вопросов](question-generation.md).
