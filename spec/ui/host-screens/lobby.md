# Host: Lobby

Экран ведущего на фазе [lobby](../../game/phases.md#lobby).
Отображается на главном экране, который виден всем участникам.

## Элементы экрана

### QR-код

QR-код со ссылкой для подключения игроков. Ссылка содержит
идентификатор комнаты и способ связи.

### Список игроков

Подключённые игроки отображаются с распределением по командам.

- В режиме **двух команд** — две колонки (красная и синяя).
- В режиме **одной команды** — одна колонка (бежевая).

Для каждого игрока отображается [PlayerAvatar](../components/PlayerAvatar.md)
и статус готовности.

### Действия ведущего

- **Перенести игрока** — перетащить игрока в другую команду.
- **Кикнуть игрока** — удалить игрока из комнаты.

### Кнопки

- **Калибровка** — открывает [попап калибровки](../../systems/audio.md).
- **Начать** — доступна, когда все игроки нажали «Готов».
  Запускает игру и блокирует подключение новых игроков.
