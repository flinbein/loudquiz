# Технические требования

## Фронтенд
В результате сборки должен быть набор статичных файлов, которые можно будет выложить на github pages или любой другой статичный хостинг.

`npm run dev` - запуск приложения в режиме отладки. Должно быть доступно в локальной сети (на всех IP)

`npm run dev:storybook` - запуск сторибука с библиотекой компонентов

## Бекенд
Без бекенда. Для связи игроков используется WebRTC.
В качестве сигнального сервера используется https://www.npmjs.com/package/p2pt
