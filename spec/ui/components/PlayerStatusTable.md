# PlayerStatusTable

Таблица состояния игроков в текущем раунде. Отображает состав
активной команды с ролями и статусами.

## Структура строки

Каждая строка содержит:

| Элемент | Описание |
|---|---|
| Аватар | [PlayerAvatar](PlayerAvatar.md), размер `small` |
| Имя | Имя игрока, цветом команды |
| Роль | Иконка или бейдж роли в раунде |
| Статус | Текущий статус действия |

## Роли

| Роль | Отображение |
|---|---|
| Капитан | Иконка короны |
| Игрок (обычный раунд) | Иконка блокнота |
| Игрок с номером (блиц) | Бейдж с номером в очерёдности |
| Не определена | Знак вопроса |

## Статусы

| Статус | Отображение |
|---|---|
| Дал ответ | Галочка |
| Нажал «Не знаю» (блиц) | Крестик |
| Печатает | Анимация набора текста |
| Ожидание | Пусто |
