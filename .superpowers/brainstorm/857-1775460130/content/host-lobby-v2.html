Изменения в PlayerStatusTable

  • Новый prop mode: "lobby" | "game" — переключает поведение
  • lobby mode: показывает ready-статус (✓/⏳) вместо role/status, кнопку kick (✕), draggable строки
  • game mode: текущее поведение (роль, статус ответа)
  • Группировка по командам: HostLobby группирует игроков в секции (красные → синие → без команды)
  • Drag: нативный HTML DnD, drag-handle на строке, drop-zone на секции команды
