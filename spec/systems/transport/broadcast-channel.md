# Broadcast Channel

Реализация [транспортного слоя](README.md) на основе
Broadcast Channel API для локальной отладки.

## Назначение

Позволяет тестировать многопользовательскую игру в пределах
одного браузера. Каждая вкладка — отдельный участник (host или player).

## Принцип работы

Broadcast Channel API обеспечивает обмен сообщениями между вкладками
одного браузера на одном домене.

- Host создаёт канал с уникальным именем (roomId).
- Player подключается к каналу по roomId.
- Все сообщения получают все участники канала.

## Ограничения

- Работает только в пределах одного браузера.
- Не работает между устройствами.
- Предназначен исключительно для разработки и отладки.

## Приоритет реализации

**Первый** — реализуется в первую очередь для обеспечения
быстрой отладки без внешних зависимостей.
