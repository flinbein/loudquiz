# Аватары игроков

Аватар — визуальное представление игрока в интерфейсе. Отображается
в лобби, таблицах статусов, стикерах и карточках.

## Внешний вид

Аватар представляет собой **кружок** с эмодзи-иконкой внутри.

- **Фон** — цвет команды игрока (красный, синий, бежевый).
- **Иконка** — эмодзи из [пула иконок](#пул-эмодзи).
- **Имя** — отображается в нижней части аватара (кроме мелкого размера).

### Формат имени

Короткое имя игрока, определяется по правилам:

1. Из имени игрока удаляются все спецсимволы (остаются только буквы).
2. Если результат содержит 2+ слов — берутся **инициалы** (первые буквы двух слов).
3. Иначе — **первые 3 буквы** имени.

## Выбор иконки

- При подключении игрока иконка выбирается **случайно** из пула,
  исключая уже занятые другими игроками иконки.
- В лобби игрок может **сменить иконку**, кликнув по своему аватару.
  Новая иконка выбирается случайно из свободных.

## Состояния

- **Online** — обычное отображение.
- **Offline** — аватарка становится **серой и полупрозрачной**.
