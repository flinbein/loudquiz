# Host: Finale

Экран ведущего на фазе [finale](../../game/phases.md#finale).
Итоговый экран с результатами игры.

## Элементы экрана

### Счёт команд

Итоговый счёт каждой команды. В режиме двух команд — выделение победителя.

### Таблица заданий

[TaskView](../components/TaskView.md) — полная таблица сыгранных заданий
с набранными баллами.

### Статистика игроков

Три номинации:

- **Лучший игрок** — дал больше всех верных ответов (не учитывая блиц).
- **Лучший капитан** — принёс команде больше всех баллов в качестве капитана.
- **Самый быстрый игрок** — в среднем отвечал быстрее всех.

Для каждой номинации отображается [PlayerAvatar](../components/PlayerAvatar.md)
победителя.

### Кнопки

- **Играем ещё раз!** — доступна только в AI-режиме.
  Переход к фазе [lobby](../../game/phases.md#lobby) в той же комнате.
  Новые игроки могут присоединиться. Использованные вопросы сохраняются
  в localStorage. Настройки игры
  ([GameSettings](../../systems/state.md#gamesettings)) сохраняются.
