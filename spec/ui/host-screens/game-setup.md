# Host: Game Setup

Экран создания игры на главной странице (`/`). Отображается после нажатия
кнопки «Новая игра». Host настраивает параметры, после чего создаётся
комната и происходит переход на `/play?room=<id>`.

## Выбор командного режима

- **Одна команда** (single mode) — кооперативный режим.
- **Две команды** (dual mode) — соревновательный режим.

## Выбор источника заданий

### Ручной режим

- **Загрузка JSON** — кнопка выбора файла
  ([формат QuestionsFile](../../tools/constructor.md#формат-файла)).
- **Предпросмотр** — таблица загруженных заданий.
- **Кнопка «Загрузить другой файл»** — замена загруженного файла.

### AI-режим

Настройки:

| Параметр | По умолчанию | Описание |
|---|---|---|
| API KEY | из localStorage | Ключ OpenRouter API |
| Количество тем | 3 | Число тем для генерации |
| Вопросов на тему | 4 | Число вопросов на каждую тему |
| Блиц-раундов на команду | 2 | Число блиц-раундов |

Список ранее использованных вопросов загружается из localStorage
и передаётся AI для исключения повторов
(см. [персистентность](../../systems/persistence.md)).

### Валидация

Кнопка «Создать игру» доступна при выполнении условия:

```
(количество_тем × вопросов_на_тему) > 0 || блиц_раундов_на_команду > 0
```

## Кнопки

- **Создать игру** — создаёт комнату, сохраняет
  [GameSettings](../../systems/state.md#gamesettings) в sessionStorage,
  переходит на `/play?room=<id>`.
