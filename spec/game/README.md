# Игровая механика

Раздел описывает правила, фазы и систему подсчёта очков в Loud Quiz.

## Содержание

- [Глоссарий](glossary.md) — единый справочник терминов
- [Правила игры](rules.md) — правила составления вопросов, оценка ответов
- [Фазы игры](phases.md) — состояния игры и переходы между ними
- [Подсчёт очков](scoring.md) — формулы начисления баллов
- [Блиц-раунды](blitz.md) — механика блиц-заданий

## Общая структура игры

Игра состоит из серии раундов двух типов:

1. **Обычные раунды** (Round) — капитан объясняет вопрос жестами,
   все игроки команды пишут ответы. Подробнее: [правила](rules.md).
2. **Блиц-раунды** (Blitz) — капитан объясняет слово первому игроку,
   тот передаёт следующему по цепочке. Подробнее: [блиц](blitz.md).

Сначала играются все обычные раунды (карточки с заданиями),
затем — блиц-раунды.
