# BlitzBox

Светло-коричневая коробка для отображения блиц-заданий
в [TaskView](TaskView.md).

## Параметры

```typescript
interface BlitzBoxProps {
  active: boolean;
  teamColor?: "red" | "blue" | "none";
  text?: string;
}
```

| Параметр | Описание |
|---|---|
| `active` | Свечение вокруг коробки, анимация покачивания |
| `teamColor` | Цвет текста: красный, синий, бежевый. Без значения — белый |
| `text` | Отображаемый текст. Если не задан — отображается «?» |

## Визуальные состояния

### По умолчанию
Коробка с текстом «?» или указанным текстом посередине.

### Активная
Свечение вокруг коробки. Небольшая анимация покачивания.

### С результатом
Вместо «?» отображается количество заработанных баллов
цветом команды.

## Использование

- [TaskView](TaskView.md) — список блиц-раундов под таблицей вопросов
