# StickerStack

Группирует несколько [Sticker](Sticker.md) в стопку.
Используется при объединении одинаковых ответов.

## Поведение

### Один стикер

Отображает стикер как есть, без дополнительного оформления.

### Несколько стикеров

- Отображает **первый стикер** сверху.
- Под ним — **подложка**, имитирующая наличие других стикеров.
- В правом верхнем углу — **бейдж** с количеством стикеров в стопке.

### Взаимодействие

- **Клик по подложке** — листает стикеры по кругу, показывает следующий.
- **Клик по бейджу** — разделяет стопку (доступно только ведущему).

## Параметры

```typescript
interface StickerStackProps {
  children: StickerElement[];
  onClick?: () => void;
  onBadgeClick?: () => void;
}
```

| Параметр | Описание |
|---|---|
| `children` | Массив компонентов [Sticker](Sticker.md) |
| `onClick` | Клик по подложке — листание стикеров |
| `onBadgeClick` | Клик по бейджу — разделение стопки (для ведущего) |
