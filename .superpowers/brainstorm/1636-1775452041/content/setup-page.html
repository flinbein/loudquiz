Логика SetupPage

  • Team Mode: toggle single/dual — сохраняется в GameSettings
  • Manual: file input (JSON) → парсинг → валидация → topics/blitzTasks в store
  • AI: API key (из localStorage), topic count, questions/topic, blitz rounds
  • «Создать игру»: сохраняет settings + topics в sessionStorage, создаёт комнату, navigate → /play?room=b-xxx
  • AI settings показываются только при выборе AI-режима
  • Валидация файла: JSON должен содержать массив topics с questions
