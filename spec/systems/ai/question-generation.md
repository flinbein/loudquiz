# Генерация вопросов

AI создаёт вопросы для обычных раундов по выбранным темам.

## Входные данные

```typescript
interface QuestionGenerationInput {
  topics: string[];
  questionsPerTopic: number;
  playersPerTeam: number;
  pastQuestions: string[];
}
```

| Поле | Описание |
|---|---|
| `topics` | Список тем |
| `questionsPerTopic` | Количество вопросов на тему |
| `playersPerTeam` | Количество игроков в команде |
| `pastQuestions` | Вопросы из прошлых игр (не повторять) |

## Правила для AI

### System prompt

Роль: составитель вопросов для викторины с наушниками.

Инструкции:
- Каждый вопрос должен иметь **не менее `playersPerTeam` уникальных ответов**.
  Синонимы не считаются уникальными.
- Ответы должны быть **проверяемыми**.
- Сложность — от 100 до 200, кратно 10. Распределение примерно равномерное.
- Учитывать, что капитан объясняет вопрос **жестами и мимикой**.
  Основной критерий сложности — насколько сложно передать смысл без звука.
- Допускается использование **исключающих условий** и **каламбуров**
  для повышения сложности.
- **Не повторять** вопросы из `pastQuestions`.
- `acceptedAnswers` — предоставить `playersPerTeam` примеров верных ответов.

### User prompt

Содержит:
- Список тем.
- Количество вопросов и игроков.
- Список прошлых вопросов (если есть).

## Результат

```typescript
interface QuestionGenerationResult {
  topics: Array<{
    name: string;
    questions: Array<{
      text: string;
      difficulty: number;
      acceptedAnswers: string[];
    }>;
  }>;
}
```
