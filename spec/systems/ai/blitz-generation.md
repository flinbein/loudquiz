# Генерация блиц-заданий

AI создаёт задания для блиц-раундов — слова и словосочетания,
которые капитан объясняет жестами.

## Входные данные

```typescript
interface BlitzGenerationInput {
  rounds: number;
  tasksPerRound: number;
  pastTasks: string[];
}
```

| Поле | Описание |
|---|---|
| `rounds` | Количество блиц-раундов |
| `tasksPerRound` | Количество вариантов заданий на раунд |
| `pastTasks` | Прошлые задания (не повторять) |

## Правила для AI

### System prompt

Роль: составитель блиц-заданий для викторины с наушниками.

Инструкции:
- Каждое задание — **слово или словосочетание**, которое нужно показать жестами.
- Сложность от 200 до 400, кратно 10.
- Распределение сложности внутри раунда — **от лёгких к сложным**.
- `text` — одновременно задание и правильный ответ.

**Критерии сложности:**
- **200–250** (легко): фрукт, животное, простое действие.
  Легко показать жестами, слово короткое.
- **260–300** (средне): профессия, предмет быта, известная личность.
- **310–350** (сложно): абстрактное понятие, длинное словосочетание.
- **360–400** (очень сложно): составное понятие с предлогом,
  малоизвестное слово.

**Ограничения:**
- Абстрактные понятия (Счастье, Демократия) — не менее 350.
- **Не повторять** слова из `pastTasks`.
- Внутри раунда варианты должны охватывать **разные уровни** сложности.

### User prompt

Содержит:
- Количество раундов и заданий.
- Список прошлых заданий (если есть).

## Результат

```typescript
interface BlitzGenerationResult {
  rounds: Array<{
    items: Array<{
      text: string;
      difficulty: number;
    }>;
  }>;
}
```
