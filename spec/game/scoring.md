# Подсчёт очков

Система начисления баллов в Loud Quiz. Очки начисляются за каждый раунд
и суммируются в общий счёт команды.

Терминология: [глоссарий](glossary.md).

## Карта джокера

У каждой команды есть **одна карта джокера** на всю игру.

- Активируется капитаном на фазе [round-pick](phases.md#round-pick).
- Удваивает (×2) баллы за раунд, в котором использована.
- Можно использовать только **один раз за игру**.

## Обычный раунд

### Все ответы верные и уникальные

Если **все** ответы верные **и** количество уникальных ответов равно
`responders_count` (ни один ответ не объединён):

```
баллы = стоимость_задания × responders_count × (1 + бонусное_время / время_раунда)
```

Где:
- `стоимость_задания` — 100..200
- `responders_count` — количество игроков в команде без капитана
- `бонусное_время` — оставшееся время round-active
- `время_раунда` — полная длительность round-active

### Условия начисления бонусного времени

Бонусное время учитывается **только** при выполнении **всех** условий:

1. Все ответы **верные**.
2. Все ответы **уникальные** (ни один не объединён).
3. Все responders ответили **во время фазы round-active** (до истечения таймера).

Если хотя бы одно условие не выполнено — `бонусное_время = 0`.

### Частично верные или объединённые ответы

Если хотя бы один ответ неверный или объединён с другим,
**бонусное время не учитывается**:

```
баллы = стоимость_задания × количество_стопок_с_верными_ответами
```

Где `количество_стопок_с_верными_ответами` — число уникальных верных ответов
после объединения одинаковых.

**Алгоритм подсчёта (AI-режим проверки):**
Каждому принятому ответу AI присваивает положительный номер группы
(поле `group`); отклонённые получают `group: null`. Количество стопок
считается как число **различных** значений `group` среди ответов
с `accepted: true`:

```
количество_стопок_с_верными_ответами = |distinct({ r.group : r.accepted === true })|
```

Подробнее о правилах группировки:
[Проверка ответов](../systems/ai/answer-check.md#уникальность-группировка).

### Джокер

Если активна карта джокера:

```
итого = баллы × 2
```

## Блиц-раунд

### Неверный ответ

Если ответ неверный или никто не ответил — **0 баллов**.

### Верный ответ

```
баллы = стоимость_задания × номер_игрока × (1 + бонусное_время / время_раунда)
```

Где:
- `стоимость_задания` — 200..400
- `номер_игрока` — порядковый номер игрока, давшего ответ, начиная с 1
  после капитана (капитан → **1** → 2 → ... → N)
- `бонусное_время` — оставшееся время blitz-active
- `время_раунда` — полная длительность blitz-active

Чем дальше по цепочке дошёл ответ — тем больше очков.

### Условия начисления бонусного времени (блиц)

Бонусное время учитывается **только** если **последний игрок** в цепочке
ответил во время фазы blitz-active (до истечения таймера).

Если ответ получен в фазе blitz-answer — `бонусное_время = 0`.

## Статистика в финале

В конце игры отображается статистика:

- **Лучший игрок** — дал больше всех верных ответов (не учитывая блиц)
- **Лучший капитан** — принёс команде больше всех баллов в качестве капитана
- **Самый быстрый игрок** — в среднем отвечал быстрее всех
