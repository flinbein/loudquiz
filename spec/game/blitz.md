# Блиц-раунды

Блиц — особый тип раунда, в котором капитан объясняет слово жестами,
а игроки передают его по цепочке. Каждый следующий игрок должен
объяснить слово следующему, не используя звук.

## Механика цепочки

```
Капитан ➡️ Игрок 1 ➡️ Игрок 2 ➡️ ... ➡️ Игрок N
```

1. Капитан видит слово и объясняет его жестами **Игроку 1**.
2. Игрок 1 понимает слово и объясняет его **Игроку 2**.
3. Цепочка продолжается до **последнего игрока**.
4. Последний игрок вводит ответ.

## Правила составления блиц-заданий

### Формат задания

Каждое блиц-задание содержит несколько вариантов слов/словосочетаний
разной сложности. Капитан выбирает одно из них.

### Сложность (200–400, кратно 10)

| Диапазон | Уровень | Примеры |
|---|---|---|
| 200–250 | Легко | Коза, Телефон, Бокс |
| 260–300 | Средне | Трансформатор, Библиотекарь |
| 310–350 | Сложно | Маленький принц, Демократия |
| 360–400 | Очень сложно | Мост в Терабитию, Квантовая физика |

### Факторы сложности

- Чем сложнее **показать жестами** — тем выше сложность
- **Узкоспециализированное слово** — повышенная сложность
- **Словосочетание** — повышенная сложность
- **Словосочетание с предлогом** — ещё сложнее
- **Абстрактные понятия** (Счастье, Демократия) — не менее 350

## Проверка ответа

Ответ в блиц-раунде проверяется **программно**, без участия AI или ведущего.

Правила нормализации:
- Без учёта регистра: `Коза` = `коза`
- `ё` и `е` эквивалентны: `Козлёнок` = `Козленок`
- Спецсимволы и пробелы игнорируются: `крем-брюле` = `крем брюле` = `крембрюле`

## Фазы блиц-раунда

### blitz-captain

Выбор капитана и очерёдности игроков. Совмещённая фаза.

- **Таймер:** 60 секунд
- **Фаза 1:** игроки нажимают «Я капитан» (первый нажавший — капитан)
- **Фаза 2:** остальные игроки выбирают слоты [1] [2] [3]...
  Слот бронируется за первым нажавшим.
- **Ограничение:** один игрок не может быть капитаном 2 раза подряд
- **Таймаут:** капитан и очерёдность назначаются случайно

Подробнее: [host](../ui/host-screens/blitz.md#blitz-captain),
[player](../ui/player-screens/blitz.md#blitz-captain)

### blitz-pick

Капитан выбирает блиц-задание.

- **Таймер:** 60 секунд
- **Действие:** капитан видит карточки с заданиями (стоимость + текст),
  выбирает одно
- **Таймаут:** задание выбирается случайно

Подробнее: [host](../ui/host-screens/blitz.md#blitz-pick),
[player](../ui/player-screens/blitz.md#blitz-pick)

### blitz-ready

Подготовка к блиц-раунду.

- **Музыка:** включается у игроков активной команды
- **Действие:** каждый игрок нажимает «Я готов»
- **Условие перехода:** все игроки готовы

Подробнее: [host](../ui/host-screens/blitz.md#blitz-ready),
[player](../ui/player-screens/blitz.md#blitz-ready)

### blitz-active

Цепочка объяснений. Таймер идёт.

- **Музыка:** играет у игроков активной команды
- **Таймер:** `10с + 20с × responders_count`
  (логика: 5с на чтение задания + 20с на каждую передачу + 5с на запись ответа,
  см. [глоссарий](glossary.md))
- **Сигнал:** звуковой сигнал + вибрация в начале и конце
- **Капитан** видит слово и текст «Объясните игроку {имя}»
- **Промежуточный игрок** видит «Вам объясняет {имя}. Объясните {имя}»
- **Последний игрок** видит поле для ввода ответа
- **Условие перехода:**
  - Последний игрок ответил → `blitz-review`
  - Таймер истёк → `blitz-answer`

Подробнее: [host](../ui/host-screens/blitz.md#blitz-active),
[player](../ui/player-screens/blitz.md#blitz-active)

### blitz-answer

Дополнительное время. Игроки отвечают по очереди, начиная с конца цепочки.

- **Музыка:** затухает (~3 сек)
- **Таймер:** `20с + 5с × responders_count`
- **Действие:** текущий игрок вводит ответ или нажимает «Не знаю»
- **«Не знаю»** — очередь переходит к следующему игроку (ближе к капитану)
- **Если все нажали «Не знаю»** — переход к `blitz-review` без ответа
- **Если кто-то ответил** — переход к `blitz-review`

Подробнее: [host](../ui/host-screens/blitz.md#blitz-answer),
[player](../ui/player-screens/blitz.md#blitz-answer)

### blitz-review

Результат блиц-раунда.

- Отображается **схема раунда** — цепочка игроков с ответом и бонусным временем
- Начисляются баллы по [формуле](scoring.md#блиц-раунд)
- Кнопка «Продолжить»

#### Схема раунда

```
[Капитан] ➡️ [Игрок 1] ➡️ [Игрок 2] ... ➡️ [Игрок N]
                              ⬇️ (+0:10)
                         [карточка ответа]
```

Схема показывает, кто дал ответ и сколько бонусного времени осталось.

Подробнее: [host](../ui/host-screens/blitz.md#blitz-review),
[player](../ui/player-screens/blitz.md#blitz-review)
