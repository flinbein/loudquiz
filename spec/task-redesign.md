# Редизайн

Нужно сделать редизайн некоторых элементов приложения.

Аватар игрока (может быть маленький, средний или большой):
- заливка цветом команды игрока. Если нет деления по командам - используй какой-то нейтральный цвет.
- Иконка эмодзи
- Поверх иконки в нижней части инициалы игрока. Или первые 3 буквы имени, если игрок ввёл 1 слово как имя. Не отображается для маленького размера.

Подготовь компонент Sticker, который будет выглядеть как бумажный стикер, у которой слегка помят уголок. В верхней части стикера горизонтальная проклеенная полоска (чуть светлее). Добавь тени для реалистичности. Если получится - сделай шершавую структуру. На стикере будет отображаться текст печатным шрифтом. Могут понадобиться стикеры разных цветов. Стикер может быть повёрнут на небольшой угол.

Стикер с аватаром игрока:
В левом верхнем углу стикера отображается аватар среднего размера в виде наклейки. Разместить аватар таким образом, будто он "Приклеивает" стикер к стене, т.е. слегка выходит за границу стикера.

Стикер с печатью:
В средней нижней части стикера ставится печать с обводкой и печатным текстом. Может быть разного цвета, с небольшой прозрачностью (выбери лучший вариант наложения прозначности, чтобы было похоже на чернильную печать). Печать может анимироваться, чтобы был эффект, будто кто-то сейчас ставит печать. 

В дальнейшем я буду называть экранами состояние приложения при определёллых условиях.
Пример: Экран `lobby_player` это игра в состоянии `lobby` для роли `player`.

## Таблица вопросов
- Карточки сделать в виде конвертов (где-то в проекте уже есть отображение конверта). Анимация разворотов карточки больше не нужна.
- На верхней части конверта писать стоимость вопроса.
- Добавить периодическую анимацию переливания на конверты
- Если какая-то команда выбрала вопрос, отобразить в левой части конверта средний аватар капитана, который выбрал этот вопрос.
- Если команда выбрала вопрос, конверт раскрасить в цвет команды. Отображать конверт как вскрытый.
- Если 100% игроков дали ответ на этом вопросе - отобразить пиктограмму звёздочки на правой части конверта.
- Если на вопросе был использован джокер, отобразить пиктограмму джокера на правой части конверта. Она может накладываться на звёздочку (с небольшим смещением)
- Если на вопрос дали ответ, на нижней части конверта писать, сколько баллов получила команда.
- Если вопрос был выбран, но все игроки дали неверный ответ - вместо количества баллов нарисовать крест.
- Если вопрос сейчас активен (текущий раунд) - отображать конверт открытым. Открывать конверт с анимацией.


## Экран lobby_player
Когда игрок подключается в лобби, сгенерировать ему иконку (случайный эмодзи).

На экране должно отображаться:
- имя (по цвету команды игрока. Белым, если играет только одна команда)
- Большой аватар. По клику игрок меняет свою иконку на случайную (но он не должен совпасть с иконкой другого игрока, поэтому выбор нового аватара происходит на бэке)
- Текст "Выберите команду"
- Если 2 команды: Кнопки "Красные", "Синие". Активная кнопка должна увеличиваться в размере и вытеснять оставшуюся кнопку, с анимацией.
- кнопка "Я только посмотрю"
- список игроков в 2 колонки. Если игроки в разных командах, они отображаются по разным колонкам. маленький аватар игрока, имя цвета команды (или без цвета, если режим для одной команды)
- кнопка "Начать игру", прибита к нижней части экрана.
- кнопка "Начать игру" сейчас сливается с фоном в светлой теме, если она не активна. Исправить.

## Экран blitz-captain-host
Слишком много пустого пространства.
Над блоком "Ждём, кто возьмёт роль капитана" нужно добавить большую картинку или SVG. Что-то, подходящее по смыслу.

## Экран blitz-pick-host
Вместо блоков с текстом "Сложности: 200 / 250 / 300" Нужно отобразить три карточки с ценами заданий на текущий раунд. Сделать в виде стикеров (цвет команды). Стикеры слегка наклонить.
Стикеры отображаются в сетке по N штук в ряд (зависит от размера экрана).
Например:
```
Капитан Алексей выбирает задание
      [200] [220] [250]
         [270] [300]
```

## Экран blitz-result_host, round-result_host
Правильный ответ отображать на карточке "Секретно", как на экране round-active_captain. 
Под карточкой - стоимость вопроса.

Ответы игроков отображать в виде стикеров. Слегка повернуть в случайную сторону. Стикеры в цвет команды.
Стикер содержит:
- Аватар игрока
- Ответ игрока печатным шрифтом
- Комментарий к ответу от ИИ мелким текстом, рукописным шрифтом
- Печать:
  - Если ответ засчитан: зелёная печать с количеством баллов, которые принёс этот игрок
  - Если ответ не засчитан: красная печать со случайным текстом "плохо", "отклонено", "неверно", "ошибка", "нет" (подбери несколько вариантов текста).
  - (blitz) Если ответ не засчитан, но верный (цепочка верных ответов прервалась ранее): черная печать с текстом "жаль", "хорошая попытка", "бесполезно" (подбери несколько вариантов текста).
  - (round) Если ответ засчитан как повтор, то он накладывается поверх карточки, которую повторяет, с небольшим отступом, как в колоде. Ведуший может кликнуть на карточку, чтобы переместить её наверх стопки.
Стикеры должны появляться по очереди с анимацией.
Печать на стикере появляется быстрой анимацией чуть позже того, как появился стикер.
Если игроки получили дополнительные баллы за преждевременный ответ, то ниже отобразить блок с информацией об этом.
Ниже - итоговый счёт за раунд.

Затем конфетти.

Затем показать статистику.

Стикер лучшего капитана: на стикере имя игрока и зелёная печать "Лучший капитан".
Отображать стикеры в сетке.
Анимировать появление стикеров, анимировать появление печатей на стикерах.

## round-pick_captain

- Цвет команды игрока

Анимировать появление стикеров.
Если стикеров слишком много - пролистывать блок со стикерами вниз, при появлении новых.
