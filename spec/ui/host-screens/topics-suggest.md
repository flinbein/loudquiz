# Host: Topics Suggest

Экран ведущего на фазе [topics-suggest](../../game/phases.md#topics-suggest).
Отображается только в AI-режиме.

## Фаза 1 — Сбор тем

### Элементы экрана

- **Таймер** — 60 секунд.
- **Список предложенных тем** — темы, которые ввели игроки, обновляются в реальном времени.
- **Список выбранных тем** — N тем, которые выбрал AI. Ведущий может:
  - Удалить тему из списка.
  - Добавить свою тему вручную.

### Кнопки

- **Ввести вручную** — останавливает таймер, ведущий вводит темы самостоятельно.
- **Продолжить** — активна, когда выбраны N тем. Переход к фазе 2.

### Переход к фазе 2

Происходит автоматически при:
- Истечении таймера (если достаточно тем).
- Нажатии «Продолжить».
- Все игроки нажали «Больше нет идей» (если достаточно тем).

## Фаза 2 — Предпросмотр заданий

### Элементы экрана

- **Таблица заданий** ([TaskView](../components/TaskView.md)) —
  вопросы, сгенерированные AI на основе выбранных тем.

### Кнопки

- **Начать игру!** — переход к первому раунду.
