Логика HostLobby

  • QR-код: генерируется через qrcode.react из joinUrl (полученного от transport)
  • Список игроков: из store.players, отображаются через PlayerAvatar + имя + статус ready
  • Drag & Drop (dual): нативный HTML DnD — перетаскивание игрока между колонками-командами → dispatch set-team
  • Kick: кнопка ✕ → удаление игрока из store + disconnect
  • Старт: disabled пока не все online + ready + min 2 игрока/команду. Tooltip с причиной
  • Калибровка: кнопка открывает CalibrationPopup (будет в Фазе 7, пока заглушка)
