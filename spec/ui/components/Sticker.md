# Sticker

Карточка с ответом игрока. Стилизована как бумажный стикер
с клеевой полоской и загнутым углом.

## Параметры

```typescript
interface StickerProps {
  player?: PlayerData;
  answerText: string;
  aiComment?: string;
  stampText: string;
  stampColor: "green" | "red";
  onClick?: () => void;
}
```

| Параметр | Описание |
|---|---|
| `player` | Игрок, давший ответ. Аватар в виде наклейки |
| `answerText` | Текст ответа, рукописный шрифт |
| `aiComment` | Комментарий AI, печатный шрифт |
| `stampText` | Текст на печати |
| `stampColor` | Цвет печати: зелёный (верно) или красный (неверно) |
| `onClick` | Клик для оценки ведущим (верно / не верно) |

## Визуальное оформление

### Стикер

- Бумажный стикер с **клеевой полоской** в верхней части.
- Нижний правый угол **загнут**.
- **Тень** под стикером.
- Стикер повёрнут на **случайный небольшой угол** (±3–5°).

### Цвет стикера

Зависит от команды игрока:

| Команда | Цвет стикера |
|---|---|
| Красная | Светло-красный |
| Синяя | Светло-синий |
| Бежевая | Светло-бежевый |

### Аватар

[PlayerAvatar](PlayerAvatar.md) (размер `small`) в левой верхней части.
Стилизован как наклейка — слегка выходит за границы стикера.

### Печать

Мокрая печать в нижней правой части стикера.
Повёрнута на случайный небольшой угол.

**Верный ответ:**
- Цвет — зелёный.
- Текст — `+{баллы}` (количество заработанных баллов).

**Неверный ответ:**
- Цвет — красный.
- Текст — случайный из: `+0`, `Плохо`, `Неверно`, `Неправильно`, `Нет`, `Отклонить`.

Печать появляется с **анимацией zoom-out**.
