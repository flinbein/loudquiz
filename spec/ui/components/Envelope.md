# Envelope

Бежевый конверт с бумагой внутри. Используется в [TaskView](TaskView.md)
для отображения вопросов обычных раундов.

## Параметры

```typescript
interface EnvelopeProps {
  open: boolean;
  label: string;
  paperText: string;
  active: boolean;
  player?: PlayerData;
  jokerUsed: boolean;
}
```

| Параметр | Описание |
|---|---|
| `open` | Состояние конверта. `true` — бумага слегка вылезает из конверта |
| `label` | Текст в нижней части конверта. Цвет — тёмно-коричневый |
| `paperText` | Текст на бумаге. Цвет зависит от команды |
| `active` | Свечение вокруг конверта, небольшая анимация покачивания |
| `player` | Игрок, выбравший этот вопрос. Аватар в левом верхнем углу |
| `jokerUsed` | Иконка джокера в правом верхнем углу |

## Визуальные состояния

### Закрытый конверт
Бумага скрыта внутри. Текст на бумаге не виден.

### Открытый конверт
Бумага слегка вылезает из конверта. Текст на бумаге виден.
Переход между состояниями — с анимацией.

### Активный конверт
Свечение вокруг конверта. Небольшая анимация покачивания.

## Цвет бумаги

Цвет бумаги и текста на ней зависит от команды игрока:

| Команда | Цвет бумаги | Цвет текста |
|---|---|---|
| Красная | Светло-красный | Тёмно-красный |
| Синяя | Светло-синий | Тёмно-синий |
| Бежевая | Светло-бежевый | Тёмно-коричневый |
