# PlayerAvatar

Аватар игрока. Кружок с эмодзи-иконкой и коротким именем.

## Параметры

```typescript
interface PlayerAvatarProps {
  size: "small" | "medium" | "large";
  playerName?: string;
  team?: "red" | "blue" | "beige";
  emoji: string;
  online: boolean;
}
```

| Параметр | Описание |
|---|---|
| `size` | Размер аватара |
| `playerName` | Имя игрока. Не отображается при `size: "small"` |
| `team` | Цвет фона: красный, синий, бежевый. Без значения — нейтральный фон |
| `emoji` | Эмодзи-иконка из [пула](../avatars.md#пул-эмодзи) |
| `online` | Статус подключения. `false` — аватар серый и полупрозрачный |

## Отображение имени

Короткое имя формируется по правилам:

1. Удаляются все спецсимволы (остаются только буквы).
2. Если 2+ слов — **инициалы** (первые буквы двух слов).
3. Иначе — **первые 3 буквы** имени.

Имя отображается в нижней части аватара.
