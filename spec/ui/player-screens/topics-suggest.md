# Player: Topics Suggest

Экран игрока на фазе [topics-suggest](../../game/phases.md#topics-suggest).
Отображается только в AI-режиме.

## Фаза 1 — Предложение тем

### Элементы экрана

- **Текст** — просьба предложить тему для викторины.
- **Текстовое поле** — ввод темы. Каждый игрок может предложить N тем
  (N задаётся в настройках).

### Кнопки

- **Больше нет идей** — блокирует интерфейс. Если все игроки нажали
  эту кнопку — немедленный переход к фазе 2 (при условии достаточного
  количества тем).

## Фаза 2 — Предпросмотр заданий

### Элементы экрана

- **Таблица заданий** ([TaskView](../components/TaskView.md)) —
  вопросы, сгенерированные AI.

### Кнопки

- **Начать игру!** — любой игрок может начать игру.
